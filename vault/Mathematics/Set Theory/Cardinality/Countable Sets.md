>[!DEFINITION] Definition: Countable Set
>
>A [set](../Sets.md) is **countable** if its [cardinality](./index.md) is [equal](Size%20Comparisons%20for%20Sets.md) to the [cardinality](./index.md) of the set of the natural numbers.
>
